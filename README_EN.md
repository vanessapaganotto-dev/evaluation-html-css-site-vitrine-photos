# Bootstrap Website Project

## Description

Static web page built with HTML5 and Bootstrap 5.  
Includes a header, a responsive navigation menu, several content sections with images and text, and a footer.

---

## Technologies Used

- HTML5  
- Bootstrap 5 (via CDN)  
- Bootstrap Icons (via CDN)

---

## Installation

Clone this repository:  
```bash
git clone <repository-url>

Open the `index.html` file in a modern web browser (Chrome, Firefox, Edge, etc.).

---

## Features

- Responsive navigation menu  
- Sections with aligned images and text  
- Bootstrap cards with titles, images, and buttons  
- Simple footer
